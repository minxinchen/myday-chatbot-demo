聯絡我們

快速部署

透過 n8n workflow，不需要後端就能串接 Pinecone、Gemini 等 AI 工具。

即時回覆

ngrok 把本機 webhook 對外公開，讓 CMS 或 demo 網頁立刻可用。

安全控制

Webhook 搭配 Basic Auth，確保只有授權的網站可以呼叫。

使用方式

  1. 在聊天視窗輸入任何 MYDAY 產品問題。
  2. 訊息會送到 n8n workflow，由 AI Agent 查詢知識庫。
  3. 回覆將顯示在聊天視窗，方便 Demo 或測試。
